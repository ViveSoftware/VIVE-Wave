How to install VIVE Wave SDK:

1. Download the unitypackage from the github.
   https://github.com/ViveSoftware/VIVE-Wave/blob/master/Installer/WaveInstaller.unitypackage

2. Import the unitypackage file in the Unity editor.

3. Select the menu on the Unity UI "VIVE/Wave Installer", press "Install or Update latest version".

Note: For "Install specific version", please copy the release version from https://github.com/ViveSoftware/VIVE-Wave/releases,
then paste the version(for example: "5.2.1-r.1") to install the specific version.
